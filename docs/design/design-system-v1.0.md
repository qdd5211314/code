# 用户管理系统 设计规范 v1.0-ui

> 基于 PRD v1.0 输出 | 设计师：李高工

---

## 1. 色彩系统

### 1.1 主色（品牌色）
| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | `#1890ff` | 主按钮、链接、选中状态 |
| Primary Hover | `#40a9ff` | 主按钮悬浮 |
| Primary Active | `#096dd9` | 主按钮按下 |

### 1.2 辅助色
| 名称 | 色值 | 用途 |
|------|------|------|
| Success | `#52c41a` | 成功提示、启用状态 |
| Warning | `#faad14` | 警告提示、待处理 |
| Error | `#ff4d4f` | 错误提示、禁用状态 |
| Info | `#1890ff` | 信息提示 |

### 1.3 中性色
| 名称 | 色值 | 用途 |
|------|------|------|
| 标题文字 | `#141414` | 一级标题、重要文字 |
| 正文文字 | `#595959` | 正文、标签 |
| 辅助文字 | `#8c8c8c` | 占位符、提示文字 |
| 禁用文字 | `#bfbfbf` | 禁用状态文字 |
| 边框色 | `#d9d9d9` | 输入框边框、分割线 |
| 背景-浅灰 | `#f5f5f5` | 页面背景、卡片背景 |
| 背景-白 | `#ffffff` | 卡片、模态框 |

## 2. 字体系统

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 24px | 600 | 32px | 页面标题 |
| H2 | 18px | 600 | 26px | 区块标题 |
| H3 | 16px | 600 | 24px | 卡片标题 |
| Body | 14px | 400 | 22px | 正文 |
| Small | 12px | 400 | 20px | 辅助说明 |

**字体族：** `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`

## 3. 间距系统

基于 `8px` 基准单位：

| 用途 | 值 | 场景 |
|------|-----|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 组件内部间距 |
| md | 16px | 区块间距、表单项间距 |
| lg | 24px | 卡片间距、表单区块间距 |
| xl | 32px | 页面内大区块间距 |
| xxl | 48px | 页面顶部/底部留白 |

---

## 4. 圆角规范

| 类型 | 值 | 用途 |
|------|-----|------|
| 小 | 4px | 按钮、输入框、标签 |
| 中 | 8px | 卡片、模态框 |
| 大 | 16px | 头像、大卡片 |

---

## 5. 阴影规范

| 层级 | 值 | 用途 |
|------|-----|------|
| Base | `0 1px 2px rgba(0,0,0,0.06)` | 卡片默认 |
| Hover | `0 4px 12px rgba(0,0,0,0.1)` | 卡片悬浮 |
| Modal | `0 6px 30px rgba(0,0,0,0.16)` | 模态框、下拉 |

---

## 6. 组件规范

### 6.1 按钮
| 类型 | 背景 | 文字 | 边框 | 用途 |
|------|------|------|------|------|
| 主按钮 | `#1890ff` | `#fff` | 无 | 核心操作 |
| 次按钮 | `#fff` | `#1890ff` | `1px solid #1890ff` | 次要操作 |
| 文字按钮 | 无 | `#1890ff` | 无 | 链接式操作 |
| 危险按钮 | `#ff4d4f` | `#fff` | 无 | 删除、禁用 |

按钮尺寸：高度 32px，左右内边距 16px，圆角 4px

### 6.2 输入框
- 高度：32px（小）/ 40px（标准）
- 边框：`1px solid #d9d9d9`，焦点时 `1px solid #1890ff` + 蓝框阴影
- 圆角：4px
- 占位符颜色：`#bfbfbf`

### 6.3 表格
- 表头背景：`#fafafa`，文字：`#141414`，字重 600
- 行高：54px
- 斑马纹：偶数行背景 `#fafafa`
- 悬停行：`#e6f7ff`

### 6.4 标签（Tag）
- 成功：背景 `#f6ffed`，文字 `#52c41a`，边框 `#b7eb8f`
- 警告：背景 `#fffbe6`，文字 `#faad14`，边框 `#ffe58f`
- 错误：背景 `#fff2f0`，文字 `#ff4d4f`，边框 `#ffccc7`
- 信息：背景 `#e6f7ff`，文字 `#1890ff`，边框 `#91d5ff`

---

## 7. 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| Mobile | < 768px | 手机竖屏 |
| Tablet | 768px-1024px | 平板 |
| Desktop | > 1024px | 桌面端（主） |

**管理后台**：桌面端优先，移动端支持核心操作

---

## 8. 页面布局规范

### 管理后台框架
- 左侧导航：宽度 200px，深色背景 `#001529`
- 顶部栏：高度 48px，白色背景，含面包屑、用户信息
- 内容区：左右各 24px 内边距，上下 16px
- 内容最大宽度：100%（填充剩余空间）
