## 四、依赖关系图

```
【后端流水线】
BE-01 数据库 → BE-02 认证 → BE-03 用户管理 → BE-07 联调
                        → BE-04 角色权限  ↗
                        → BE-05 个人信息 ↗
              BE-06 操作日志 ↗

【前端流水线】
FE-01 搭建 → FE-02 登录页 → FE-03 列表页 → FE-09 联调
                         → FE-04 新增编辑 ↗
                         → FE-05 角色页  ↗
                         → FE-06 个人中心 ↗
                         → FE-07 日志页  ↗
              FE-08 移动端适配（并行）

【测试流水线】
QA-01 用例 → QA-02 功能测试 → QA-03 性能安全 → QA-04 回归

【上线】
M4 测试完成 → OP-01 环境准备 → OP-02 上线部署 → M5 上线完成
```

### 风险升级机制
- P2 风险：管总协调处理，周报同步
